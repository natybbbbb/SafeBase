# Security Policy

## Supported Versions
We support only the latest release and the main branch.

## Reporting a Vulnerability
If you discover a security issue, please email security@safebase.example with steps to reproduce, affected components, and impact. We aim to acknowledge within 72 hours.

## Scope
Smart contracts, deployment scripts, GitHub Actions workflows, and configuration that impacts funds or sensitive data.

## Disclosure
Please follow responsible disclosure. Do not create a public issue before we confirm the fix is deployed.
